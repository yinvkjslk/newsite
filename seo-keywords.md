## Общие рекомендации по использованию

### Для Title (заголовок страницы)
- Длина: 50-60 символов
- В начале: ключевое слово
- На конце: название компании или города
- Формат: [Основное ключевое слово] | [Второстепенное] — [Название]

### Для Meta Description
- Длина: 150-160 символов
- Призыв к действию
- Уникальное торговое предложение
- Контактные данные (опционально)

### Для H1
- Один на странице
- Содержит основное ключевое слово
- Не более 70 символов

### Структура ключевых слов
- ВЧ (высокочастотные): 1000+ запросов/мес — конкуренция высокая
- СЧ (среднечастотные): 100-1000 запросов/мес — средняя конкуренция
- НЧ (низкочастотные): <100 запросов/мес — низкая конкуренция

### LSI-слова
- Используются в текстах для повышения релевантности
- Не являются прямыми ключевыми словами
- Помогают поисковым системам лучше понять контент
